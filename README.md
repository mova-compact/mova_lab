# MOVA Lab

This repository is the public site for MOVA Lab and its work around:

- AI consulting for business processes
- contract AI products
- Hermes-based specialized assistants
- MOVA as the language and contract foundation

## Languages

- English
- Українська

## Local preview

1. Install Ruby and Bundler
2. Run `bundle install`
3. Run `bundle exec jekyll serve`
4. Open `http://localhost:4000/mova_lab`

## Structure

- `/` — English home
- `/uk/` — Ukrainian home
- `pages/` — static pages
- `_projects/` — product and case-study pages
- `_story/` — story chapters
- `_posts/` — articles
